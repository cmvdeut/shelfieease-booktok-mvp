# Canva carrousels — copy-paste per slide

Geen ChatGPT nodig. Werkwijze:

1. Canva → **1080 × 1350** (Instagram 4:5)
2. Achtergrond `#0A0A0A`, tekst wit, font **Montserrat Bold**
3. Plak slide-tekst hieronder → **pagina dupliceren** voor volgende slide
4. Download **PNG, alle pagina’s**
5. Instagram → Post → **meerdere foto’s** in volgorde
6. Caption: zie `INSTAGRAM-JUNI-2026-QUEUE.md` (zelfde datum) of hieronder

---

## Huisstijl (alle carrousels)

| Instelling | Waarde |
|------------|--------|
| Formaat | 1080 × 1350 px |
| Achtergrond | #0A0A0A |
| Tekst | #FFFFFF, max 2–3 regels, gecentreerd |
| Accent (optioneel) | #4ADE80, 1 regel per slide |
| Slide 1 | Geen prijs, geen app-promo |
| Laatste slide | `Sla op 📌` + klein `shelfieease.app` |

Canva-zoektermen: `instagram carousel dark minimal`, `bookstagram carousel`.
